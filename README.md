PLASTIC is an experimental automated Meme Market Maker (A-MMM) simulation that moves beyond the classic constant‑product model. It introduces a two‑dimensional state (Memory, Momentum) that jointly determine meme-price, creating a system where:

- **Price responds to trading activity**, not just direction.
- **Slippage is path‑dependent**: the cost of a trade is the integral of price along the trade’s trajectory.
- **Spread** is derived from price sensitivity adding a risk premium.
- **Inventory** affects only local quote bias, never corrupts the global price function.
- **Liquidity** acts as a hard constraint to prevent insolvency.

The result is an Automated Meme Market Maker that behaves like a coupled dynamical system with memory and momentum—a market where price rises with activity and falls when activity fades, rather than reacting directly to buy/sell direction.Which is suitable for Meme Markets.
